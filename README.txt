Cómo Compilar y Ejecutar el Main.c:
- Descargar los archivos (Main.c - List.h - List.c - Canciones.csv)
  y mantenerlos en una misma carpeta (Nombre Predeterminado: Tarea-1-Andromeda).
- En Visual Studio Code, abrir una terminal en Git Bash.
- Utilizar el siguiente comando: "gcc *.c -o Tarea-1-Andromeda".
- Luego ejecutar el siguiente comando: "./Tarea-1-Andromeda".
